# Chromium Android - Fingerprint Protection

Chromium modificado con protección anti-fingerprinting para Android.

## Protecciones incluidas:
- Canvas 2D fingerprint spoofing
- toDataURL fingerprint spoofing
- ClientRects noise
- Font metrics noise

## Compilar
El APK se compila automáticamente via GitHub Actions.

## Uso
Cada instalación genera un seed único que persiste.
